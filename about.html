About

Status dashboard is a project developed by Afonso de Mori.

It's a frontend panel to consume the data supplied by Uptime Robot's API.

The project is developed in simple HTML and JavaScript (jQuery) so it can be hosted on Github Pages and "never" get down :)

You can see/clone/fork this project on Github.
